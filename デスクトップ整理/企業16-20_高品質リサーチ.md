## 【企業16】 日立コンサルティング
### 1. 企業概要
**企業規模・歴史:**
- 設立：2002年（日立製作所コンサルティング部門が独立、2003年「エクサージュ」→2006年現社名へ）
- 親会社：日立製作所（100%出資）
- 従業員数：541名（2025年2月末時点）
- 資本金：24億円
- 本社：東京都千代田区麹町

**事業内容:**
- 戦略コンサルティング：事業戦略、事業企画、業務改革
- クライアント構成：日立グループ以外が5割以上
- 対応業界：製造、流通、サービス、金融、保険、官公庁、自治体、電気、ガス、水道

### 2. 成長戦略
- **親会社主導の戦略:** 「Inspire 2027」で「Lumada売上収益比率80%、Lumada Adj. EBITA率20%」を目標
- **デジタル・グリーン・イノベーション** を成長ドライバーとして設定
- **中期経営計画:** ポートフォリオ改革加速、キャッシュフロー強化に注力
- **事業計画策定支援:** 企業のビジョンを戦略・施策に構造化する支援サービスを強化

### 3. ポジション詳細
**キャリアパス（6段階）:**
- アナリスト：プロジェクトメンバー、基礎学習中心
- コンサルタント：サブチームリーダー、独立領域担当
- シニアコンサルタント：チームリーダー、メンバー育成開始
- マネージャー・シニアマネージャー・ディレクター・シニアディレクター

**職務内容:**
- DX・IT戦略・ガバナンス施策の立案・実行支援
- 事業革新・業務生産性向上を目的としたDX/システム導入支援
- 製造業向け：スマートファクトリー、サプライチェーン改革、エンジニアリングチェーン改革

**ポストコンサル:** 事業会社、スタートアップ、ファンド業界、他ファームへの転職

### 4. 求める人材
**人物像:**
- さまざまなパートナーとともに新しい価値を提供する「イノベーター」
- ステークホルダーに働きかけ、周囲を巻き込める推進力
- 顧客成功への高いコミットメント
- 新しい挑戦への意欲

**必須スキル:**
- 4大卒以上
- PCスキル（PowerPoint、Excel、Word）
- ドキュメンテーション能力
- 論理的思考・コミュニケーション能力

**経験要件:**
- 社会人経験2年以上
- 応募職種に関連する業務経験
- キャリアの深掘り面接で、保有経験の顧客価値への結びつきを説明可能

### 5. Openwork評判
**総合評価:** 3.7点（22人の正社員回答、エン カイシャ調査）

**残業:**

**プラスTOP3:**

### 1. 企業概要
**企業規模・歴史:**
- 設立：2021年（シンプレクス・グループ傘下）
- 前身：シンプレクス・コンサルティング
- 従業員数：103名（2023年3月31日時点）
- 事業基盤：シンプレクス・グループのエンジニアリング力を活かした「ハイブリッド・コンサルティング」

**事業内容:**
- デジタルトランスフォーメーション（DX）戦略構築
- ビジネス・IT戦略策定
- 組織変革・業務改善
- システム化構想策定
- システム開発プロジェクトマネジメント
- 各種実行フェーズの伴走支援（独自の強み）

**対応業界：**
- 金融領域：証券、銀行、保険
- 非金融セクター：メディア・エンターテインメント、製造、建設、情報通信

### 2. 成長戦略
- **事業規模拡大:** 従業員数を継続的に拡大（2023年103名→2024年以降の増員計画）
- **「ハイブリッド支援」の差別化:** コンセプト→設計→実行→開発まで一気通貫支援
- **業界横展開:** EC、エンターテインメント、製造など多領域での新規案件獲得
- **新規ソリューション開発:** データサイエンスを活用した高度なコンサルティング

### 3. ポジション詳細
**コンサルタント職の主要活動:**
- プロジェクト管理：案件企画、管理、推進
- 営業活動：ディール・シェイピング、提案書作成
- 組織運営：採用、人材配置、オペレーション規定策定
- 新規ソリューション開発：システム設計、顧客ニーズの具現化

**チーム構成:**
- 経営陣：大手コンサルティングファーム出身者が中心
- メンバー：シンプレクス・グループのエンジニア＋ビジネスコンサルタント
- マネジメント：組織の成長に伴う管理職ポジション増加中

**キャリア特性:**
- スタートアップ段階での組織運営参画機会
- 業務とIT技術を統合したコンサルティング経験
- 小規模組織での高い裁量権

### 4. 求める人材
**求める人物像:**
- 成長への動機付けと自己改善能力
- 好奇心から主体的に問題解決へ取り組める人
- 任された責務への高い責任感
- 従業員とクライアントの両方の成長環境構築力
- ダイバーシティ受け入れの柔軟性

**必須経験:**
- 大手コンサルティングファームでの3年以上のプロジェクト経験
- 企画・実行リーダーシップの実績（内部・外部両面）
- 過去の業種経験は不問（全業界対応可能な人材配置）

**スキル要件:**
- DX、AI、IoT関連プロジェクト経験が特に有価
- 戦略・ビジネス領域での経験（テクノロジー軸が重視）
- 学歴より実務経験を優先評価（スタートアップ的採用姿勢）

## 【企業18】 クオンツ・コンサルティング
### 1. 企業概要
**企業規模・歴史:**
- 設立：2023年10月
- 親会社：クオンツ総研ホールディングス（旧M&A総研ホールディングス、東証プライム上場）
- 従業員数：200名超（設立から2年で達成、2026年9月期300名体制計画）
- 成長軌道：5年目に1,000名規模を目指す

**事業内容:**
- 経営戦略コンサルティング：M&A戦略、事業戦略策定
- テクノロジーコンサルティング：DX・IT戦略、AI等の最先端技術活用、PMOやガバナンス変革

**設立背景:**
- M&A仲介事業を通じて日本企業の業務・経営効率課題を強く認識
- 多数の子会社・孫会社保有企業のIT/DX化遅れに対する専門的支援

### 2. 成長戦略
**「実益主義の伴走」方針:**
- 日本企業の価値向上を最優先
- 時価総額・株価向上に直結する支援を実現

**3軸統合による差別化:**
- 「戦略」「IT/DX」「金融」の有機的統合
- 他ファームにはない特徴：M&A買収後統合（PMI）での一気通貫支援

**人員拡大計画:**
- 2023年10月：設立
- 2024年度：従業員数大幅増加
- 2026年9月：300名体制を目指す
- 中期：500名、5年目1,000名規模へ

**組織構造:**
- ワンプール制採用：全員が「人材プール」に属し、プロジェクトごとにアサイン
- 特定専門部署を廃止し、柔軟な人員配置を実現

### 3. ポジション詳細
**プロジェクト構成:**
- EC事業成長戦略の策定支援
- エンターテインメント業界の新規事業企画
- 製造業向けDX支援
- データサイエンス活用型コンサルティング

**職務内容:**
- 経営課題の分析・戦略立案
- IT/DX導入による実行支援
- M&A戦略立案から買収後統合（PMI）支援
- 組織変革・業務改善サポート

**キャリア特性:**
- 新進気鋭ファームでの急速な組織成長経験
- M&A仲介知見を活かした金融・経営支援の複合領域
- ワンプール制による多業界・多機能経験機会

### 4. 求める人材
**採用時の重視ポイント:**
- パーソナリティを最重要視
- スキル・資格は問わない（ただし実務能力は評価対象）
- 論理的な仮説立案と本質的解決策思考の有無を見極め

**必須スキル:**
- 高度な情報分析能力
- デリバリー業務経験（プロジェクト推進実績）
- 論理的思考力
- クライアント・コミュニケーション能力
- 実行力

**転職者適性:**
- 営業・企画職経験者も活躍可能
- 課題解決型業務経験があれば十分
- コンサル業界経験者比率75%（マンツーマンOJT強化）

**教育体制:**

## 【企業19】 ボストン コンサルティング グループ（BCG）
### 1. 企業概要
**企業規模・歴史:**
- 設立：1963年（米国ボストン、ブルース・ヘンダーソン創業）
- グローバル規模：世界50ヶ国、90以上の都市に拠点
- 従業員数：約30,000人（グローバル）
- 日本進出：1966年（世界2番目の拠点として東京オフィス開設）

**日本での展開:**
- 東京、名古屋（2003年開設）、大阪（2020年）、京都（2020年）、福岡（2022年4月）に拠点
- 日本マーケットでの地位：戦略ファームNo.1

**事業内容:**
- 経営戦略コンサルティング
- 支援領域：全社戦略、マーケティング・営業戦略、新規事業戦略、ポートフォリオ改革、M&A、組織改変、財務戦略、IT戦略
- 変革推進、組織力向上、競争優位性構築、収益改善が中心

### 2. 成長戦略
**デジタル先制：**
- BCG Digital Ventures、Platinion、Gamma等の専門組織を立ち上げ
- 従来の「戦略コンサルタント」採用だけでなく、アナリティクス、エンジニア、デザイナーの採用拡大

**人材の多様化：**
- プロダクト開発を自社で実施
- 開発まで行う包括的コンサルティングへのシフト

**業界・地域戦略：**
- 自動車、ハイテク、金融、医薬品を中心とした産業支援継続
- 西日本地域への拠点拡大（関西オフィスを中心に）

**グローバルトランスフォーメーション：**
- クライアントの「トランスフォーメーション全般」をカバーする統合的支援

### 3. ポジション詳細
**キャリアパス（成果主義・メリットクラシー重視）:**
- コンサルタント（新卒・経験者向け）
- プロジェクトリーダー
- マネージャー
- パートナー

**職務内容:**
- 戦略立案：クライアント企業が直面する課題への分析・洞察提供
- データ駆動型問題解決
- 実装支援：提言の実行支援まで範囲を拡大

**昇進速度:**
- 1レベル：約3年（長くて3.5年）
- 実績主義で個人差大きい

**職務の特性:**
- クライアントトップマネジメントとの直接対話機会
- グローバルプロジェクトへの参画機会
- 業界最前線のトランスフォーメーション経験

### 4. 求める人材
**BCGが求める3つの資質：**

1. **インテレクチュアル（知識的能力）**
   - しつこく考える力
   - 大胆な発想力
   - 本質を深く洞察する力

2. **インターパーソナル（対人スキル）**
   - クライアント課題への深い理解
   - ロジックと熱意を兼ね備えた説得力
   - 信頼構築能力
   - クライアントとの苦楽を共にする姿勢

3. **成長力（学習意欲）**
   - 自らの課題を素直に受け入れる力
   - 挑戦を続ける意志
   - 着実に成長する力

**必須スキル:**
- 的確な質問ができる好奇心
- 新方向性を打ち出す勇気と創造性
- 同僚・クライアントとの連携能力
- アイデアを行動に移すリーダーシップ

**転職者に求められる経験：**
- 営業で顧客課題を引き出しソリューション提案した経験
- 工場・組織全体の生産性向上プロジェクト経験
- 会社全体をビジネス俯瞰しながらプロジェクト推進経験
- **提案型スキル** が必須

**採用方針:**
- 荒削りでもポテンシャル高い人材を優遇
- 完成した人間よりも、成長意欲が強い人を評価

## 【企業20】 マッキンゼー・アンド・カンパニー
### 1. 企業概要
**企業規模・歴史:**
- 設立：1926年（米国シカゴ、James O. McKinsey創業）
- グローバル規模：世界60ヶ国、105以上の支社
- 年間プロジェクト数：1,600件以上
- 日本進出：1971年（初のアジア拠点として東京オフィス開設）

**日本での地位:**
- 日本トップ30社の8割をサポート
- 過去5年間：1,000プロジェクト以上実施
- 日本マーケットでの地位：戦略ファームNo.1水準

**日本での拠点:**
- 東京本社
- 関西オフィス（2018年3月開設）

**事業内容:**
- 全社事業戦略、事業部門戦略の構築支援
- M&A戦略の構築から実行支援
- グローバル経営戦略構築
- トランスフォーメーション全般

### 2. 成長戦略
**成長を最重要テーマに位置付け：**

3つの成長創出領域：
1. 既存コア事業の強化
2. 周辺事業領域への拡大
3. 新規事業の創出

**M&Aを成長加速の手段に活用：**
- 成長企業は成長しない企業比で約85%多くデジタル関連M&Aを実施
- M&A戦略から買収後統合（PMI）までの一気通貫支援

**デジタル・トランスフォーメーション対応：**
- クライアント企業のDX推進を支援
- マッキンゼー自体もデジタル先進企業として機能

**グローバル市場での日本重視：**
- アジアで最重要市場としての地位確立
- 西日本地域（関西オフィス経由）の事業拡大

### 3. ポジション詳細
**キャリアパス（成果主義・メリットクラシー）:**

**新卒・経験浅い層:**
- ビジネスアナリスト（最初のキャリア開始点）
- 情報収集・分析での個別分野担当
- 顧客メンバーとの協業
- 最終提案作成・プレゼンテーション参加

**中堅層:**
- アソシエイト
- マネージャー
- プロジェクト統括責任
- メンバー育成

**上位層:**
- パートナー
- トップマネジメントへのアドバイザリー

**昇進速度:**
- 1レベルに3年程度が目安
- 何歳か、何年在籍したかではなく「何を成し遂げたか」で評価（成果主義）
- 個人差が大きい（優秀者は加速）

**職務の特性:**
- データ駆動型分析を基盤
- クライアント企業トップマネジメントとの対話
- 複合領域的課題への問題解決
- グローバル案件への参画機会

### 4. 求める人材
**マッキンゼーが求める6つの観点：**

1. **Personal Impact（対人影響力）**
   - 相手に影響を及ぼす力

2. **Entrepreneurial Drive（起業家的精神）**
   - 主体的な行動意欲
   - リーダーシップ引き受ける意志

3. **Inclusive Leadership（包摂的リーダーシップ）**
   - 多様な視点を受け入れる力
   - チーム生産性向上を導く力

4. **Courageous Change（変化への果敢な対応力）**
   - 既存概念を打破する勇気
   - 自由で独自の発想力

5. **Problem Solving（問題解決能力）**
   - 課題の構造化
   - 重要論点の特定
   - データ分析能力
   - 仮説思考
   - 明快な意思決定

6. **Expertise（専門性・知見）**
   - 特定業界・領域での深い知見
   - 実務経験

**必須資質：**
- 顧客企業バリュー向上への共感と責任感
- 既存考え方にとらわれない自由な発想
- 具体的・概念的事象両方への高い理解能力
- 強い好奇心と粘り強さ
- 確かな判断力

**必須スキル：**
- 論理的思考力（課題構造化、重要論点特定）
- データ分析能力
- 仮説思考
- 意思決定能力
- 対人スキル（他者の意見を理解し、適切に返答可能）
- リーダーシップスキル

**転職者に求められるスキル：**
- 営業で顧客課題を引き出してソリューション提案した経験
- 工場・組織全体の生産性向上プロジェクト経験
- 会社ビジネス全体を俯瞰しながらプロジェクト推進経験
- **提案型の実務経験** が必須
- 特定業界・領域での深い専門知見

### 5. Openwork評判
**総合評価:** 4.1点（43人の正社員回答）

**年収:**
- 平均年収：1,231万円～1,300万円～1,800万円（幅あり）
- 業界平均比：大幅に上回る
- 新卒初任給（ビジネスアナリスト）：記載なし

**残業:**
- 月間平均残業時間：70時間程度
- プロジェクト依存で大きな変動
- 役職なしメンバー：残業3時間程度に抑えられることも
- 定時帰社が可能なプロジェクトも存在

**働き方の特徴:**
- 数ヶ月単位プロジェクトへのアサイン
- 「Take Time」：5～10週間の長期休暇制度が存在
- メリハリのある働き方が浸透

**有給消化率:** 79.4%（業界内で高い水準）
- ベイン・アンド・カンパニー比：71.5%
- ボストン・コンサルティング比：63.0%

**プラスTOP3:**
1. 年収水準が高い（1,231万円～1,800万円）
2. 有給消化率が79.4%と業界最高水準
3. 「Take Time」による長期休暇制度が充実

**マイナスTOP3:**
1. 月70時間程度の残業が常態（プロジェクト依存で変動）
2. ワークライフバランスは優れているが、激務期間の存在
3. 成果主義による評価プレッシャー

**有給消化率:** 79.4%
